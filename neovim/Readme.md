# For learning docker
- Setup neovim with configuration of [huytd](https://github.com/huytd/vim-config)
## Installation
- Install docker
- runing docker with repo
    ```
    docker run --ti --rm -v path/to/folder:/Workspace  dthtien/neovim:firsttry
    ```
